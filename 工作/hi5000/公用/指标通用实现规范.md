### 1.6 月/周/多天计算自动支持

**框架自动处理**（无需子类实现）：
- `calculateWeek()` 和 `calculateMonth()` 由框架自动实现
- 框架会：
  1. 从日数据的 `detail` 字段反序列化出 `IndicatorDetail`
  2. 累加所有日的 `qualifiedCount` 和 `totalCount`
  3. 计算汇总后的 `value = qualifiedCount / totalCount × 100`
  4. 调用 `extendTimeRangeDetail()` 让子类扩展额外字段（如果有）
  5. 设置汇总结果

**extendTimeRangeDetail()** - 扩展月/周报表字段：

```java
@Override
protected IndicatorDetail extendTimeRangeDetail(IndicatorDetail baseDetail, 
                                                    List<TBusinessIndicators> detail, 
                                                    Map<String, Object> param) {
    // baseDetail已包含汇总后的qualifiedCount和totalCount
    // 创建子类实例来扩展额外字段
    XxxMonthDetailDto extended = new XxxMonthDetailDto();
    extended.setQualifiedCount(baseDetail.getQualifiedCount());
    extended.setTotalCount(baseDetail.getTotalCount());
    
    // 添加额外字段
    int assessmentDays = (int) detail.stream()
            .filter(i -> Objects.equals(ExaminationStatus.EXAMINED.getCode(), 
                                        i.getExaminationStatus()))
            .count();
    extended.setAssessmentDays(assessmentDays);
    
    return extended;
}
```

### 2.3 detail解析示例（使用JSON反序列化）

```java
@Override
protected void fillExtraFields(TBusinessIndicators source, IndicatorQueryVo target) {
    if (source == null || target == null) {
        return;
    }
    
    // 解析detail JSON字符串为IndicatorDetail
    if (StrUtil.isNotEmpty(source.getDetail())) {
        try {
            IndicatorDetail detail = JSON.parseObject(source.getDetail(), IndicatorDetail.class);
            target.getExtra().put("合格数", detail.getQualifiedCount());
            target.getExtra().put("总数", detail.getTotalCount());
        } catch (Exception e) {
            log.warn("解析detail失败: {}", source.getDetail(), e);
        }
    }
    
    // 解析考核分
    if (StrUtil.isNotEmpty(source.getAssessmentData())) {
        target.getExtra().put("考核分", source.getAssessmentData());
    }
}
```

### 6.3 生成时间列表

```java
private List<String> buildTimeList(String formattedDate) {
    List<String> timeList = new ArrayList<>();
    if (formattedDate.equals(LocalDateTime.now().format(FORMATTER))) {
        // 今天：只计算到当前分钟
        LocalDateTime now = LocalDateTime.now();
        LocalDateTime startOfDay = LocalDate.now().atStartOfDay();
        Duration duration = Duration.between(startOfDay, now);
        long minutes = duration.toMinutes();
        timeList = Arrays.asList(BizConstant.DayTime.dayTime).subList(0, (int) minutes);
    } else {
        // 历史日期：全天1440分钟
        timeList = Arrays.asList(BizConstant.DayTime.dayTime);
    }
    return timeList;
}
```
