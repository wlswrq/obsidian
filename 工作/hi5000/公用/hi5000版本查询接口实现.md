# 版本查询接口实现

## 概述
版本查询接口用于返回系统的版本号和构建时间信息，便于运维和版本管理。

## 技术实现

### 1. 接口定义
**文件位置**: `hi5000-platform/hi5000-query/src/main/java/com/heli/hi5000/query/controller/VersionController.java`

```java
@RestController
@RequestMapping("/system")
public class VersionController {
    @PostMapping("/version")
    public HLResponse<String> getVersion() {
        return HLResponse.ok(versionService.getVersionInfo());
    }
}
```

**接口路径**: `POST /system/version`

### 2. 服务层
**接口**: `hi5000-platform/hi5000-query/src/main/java/com/heli/hi5000/query/service/VersionService.java`

**实现**: `hi5000-platform/hi5000-query/src/main/java/com/heli/hi5000/query/service/impl/VersionServiceImpl.java`

```java
@Service
public class VersionServiceImpl implements VersionService {
    @Value("${app.version:unknown}")
    private String appVersion;

    @Value("${app.build.time:unknown}")
    private String buildTime;

    @Override
    public String getVersionInfo() {
        return appVersion + "(" + buildTime + ")";
    }
}
```

### 3. 配置

#### Maven配置 (pom.xml)
**文件**: `hi5000-xinjiang/hi5000-xinjiang-portal/pom.xml`

```xml
<properties>
    <maven.build.timestamp.format>yyyy-MM-dd HH:mm:ss</maven.build.timestamp.format>
</properties>

<build>
    <resources>
        <resource>
            <directory>src/main/resources</directory>
            <filtering>true</filtering>
        </resource>
    </resources>
    <plugins>
        <plugin>
            <groupId>org.codehaus.mojo</groupId>
            <artifactId>build-helper-maven-plugin</artifactId>
            <version>3.4.0</version>
            <executions>
                <execution>
                    <id>timestamp</id>
                    <phase>validate</phase>
                    <goals>
                        <goal>timestamp-property</goal>
                    </goals>
                    <configuration>
                        <name>build.timestamp</name>
                        <pattern>yyyy-MM-dd HH:mm:ss</pattern>
                        <timeZone>GMT+8</timeZone>
                        <locale>zh_CN</locale>
                    </configuration>
                </execution>
            </executions>
        </plugin>
    </plugins>
</build>
```

#### 应用配置 (application.yml)
**文件**: `hi5000-xinjiang/hi5000-xinjiang-portal/src/main/resources/application.yml`

```yaml
app:
  version: HI-V1.0.0
  build:
    time: ${build.timestamp}
```

**JWT跳过认证配置**:
```yaml
org:
  my:
    jwt:
      skip-auth-urls:
        - /system/version
```

## 工作原理

1. **构建时**: Maven的`build-helper-maven-plugin`插件在`validate`阶段生成`build.timestamp`属性
2. **资源过滤**: `<filtering>true</filtering>`确保配置文件中的占位符`${build.timestamp}`被替换为实际构建时间
3. **运行时**: Spring通过`@Value`注解注入配置值
4. **接口返回**: 格式为 `版本号(构建时间)`

## 返回示例

```
HI-V1.0.0(2026-04-09 10:15:30)
```
