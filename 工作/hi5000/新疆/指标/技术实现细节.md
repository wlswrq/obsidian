# 技术实现细节

## 架构设计

### 整体架构

```
XJAvailablePowerTask (业务层)
    ↓ 调用
AdjustmentDataQueryService (服务层)
    ↓ 实现于
AdjustmentMapper (数据访问层)
    ↓ 查询
TDengine (时序数据库)
```

### 核心组件

1. **XJAvailablePowerTask** - 新疆可用功率计算任务
   - 负责业务逻辑编排
   - 协调各组件完成计算

2. **AdjustmentDataQueryService** - 遥调数据查询服务
   - 提供数据查询接口
   - 统一查询逻辑

3. **AdjustmentMapper** - 数据映射层
   - SQL查询实现
   - 数据实体映射

```sql
-- 主表
CREATE TABLE adjustment (
```

## 查询优化

### SQL优化策略

#### 1. 使用窗口函数

```sql
-- 修改前：UNION ALL方式
SELECT _wstart AS dataTime, ...
FROM adjustment
WHERE src_ip = '192.168.1.1' AND src_port = 8080 ...

UNION ALL

SELECT _wstart AS dataTime, ...
FROM adjustment
WHERE src_ip = '192.168.1.2' AND src_port = 8080 ...
```

```sql
-- 修改后：直接查询
SELECT _wstart AS dataTime, ...
FROM adjustment
WHERE point_table_number IN (1001, 1002, ...)
PARTITION BY point_table_number
INTERVAL(60s) FILL(PREV)
```

#### 2. 索引优化

```sql
-- 创建组合索引
CREATE INDEX idx_adjustment_point_time 
ON adjustment(point_table_number, ts);

-- 创建IP端口索引
CREATE INDEX idx_adjustment_channel 
ON adjustment(src_ip, src_port, dest_ip);
```

### 查询性能对比

| 查询方式 | 执行时间 | 数据量 | 资源消耗 |
|----------|----------|--------|----------|
| **UNION ALL** | ~5s | 86,400点 | 高 |
| **窗口函数** | ~0.5s | 1,440点 | 低 |

## 错误处理

### 异常分类

```java
// 1. 数据异常
public class DataException extends RuntimeException {
    public DataException(String message, Throwable cause) {
        super(message, cause);
    }
}

// 2. 计算异常
public class CalculationException extends RuntimeException {
    public CalculationException(String message, Throwable cause) {
        super(message, cause);
    }
}

// 3. 系统异常
public class SystemException extends RuntimeException {
    public SystemException(String message, Throwable cause) {
        super(message, cause);
    }
}
```

### 异常处理流程

```java
try {
    // 1. 数据查询
    Map<String, TreeMap<String, BigDecimal>> agcData = 
        adjustmentDataQueryService.queryAdjustmentDataRawNoChannel(...);
    
    // 2. 数据处理
    IndicatorResult result = calculateAvailablePower(...);
    
    // 3. 结果保存
    saveResult(result);
    
} catch (DataException e) {
    log.error("数据查询失败", e);
    throw new BusinessException("数据查询失败", e);
} catch (CalculationException e) {
    log.error("计算过程失败", e);
    throw new BusinessException("计算过程失败", e);
} catch (SystemException e) {
    log.error("系统异常", e);
    throw new BusinessException("系统异常", e);
}
```
