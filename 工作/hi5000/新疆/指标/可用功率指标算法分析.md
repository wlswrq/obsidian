# 可用功率指标算法分析

> 需求文档: 新疆HI5000新能源监测预警系统需求规格说明书(光伏电站)V1.0.0-20260402
> 附件一(xmind): 2025版新疆考核数据计算逻辑-20260402.xmind
> 日期: 2026-04-08

---

## 一、指标定义

- **指标名称**: 可用功率（可用电量日准确率）
- **所属模块**: 数据报表 → 考核-风光功率预测
- **核心公式**: 准确率 = 实际电量 / 可用电量 × 100%
- **合格标准**: 97%（可配置）
- **数据颗粒度**: 1秒钟1个点（计算积分电量）
- **计算频率**: 当月每日计算
- **查看逻辑**: 次日查看前日

### 关键约束

> "限电时段不参与可用功率准确率计算"

需要做**限电判断**，排除限电时段后计算准确率。

## 三、算法推导

### 3.1 数据来源

```
实发功率(KW) = |Σ(集电线路有功功率) + Σ(储能设备有功功率)|  ← 取绝对值
可用功率(KW) = 可用发电电力功率
AGC目标值    = AGC有功功率目标值（遥调数据）
装机容量     = 自定义字段配置值
```

### 3.2 限电判断（每分钟）

```
限电 = AGC目标值 < 装机容量
非限电 = AGC目标值 >= 装机容量
```

来源: 附件一+附件二确认。AGC指令缺失时沿用上一个指令值。

### 3.3 秒级电量计算

```
每秒实发电量(KW·h) = 实发功率(KW) / 3600
每秒可用电量(KW·h) = 可用功率(KW) / 3600
```

### 3.4 日准确率计算

```
非限电时段实际电量 = Σ(所有非限电秒的实发电量)
非限电时段可用电量 = Σ(所有非限电秒的可用电量)
准确率 = 非限电时段实际电量 / 非限电时段可用电量 × 100%
```

**关键**: 只有非限电时段参与计算

### 3.5 考核分计算（附件一已修正）

```
IF 准确率 < 97%:
    diff = 97 - 准确率
    考核分 = diff × 装机容量(MW) × 0.002
    单位: 分/万千瓦
ELSE:
    考核分 = 0
```

> [!warning] 与当前实现不同
> - 当前: `calcAvcOperationRateScore(diff)` = `diff × 2`
> - 新公式: `diff × 装机容量 × 0.002`（类似上报率考核 `calcReportingRateScore`）

### 3.6 月考核分

```
月考核分 = Σ(当月每日考核分)
```

## 六、附件一确认（xmind原文）

附件一内容来自 `2025版新疆考核数据计算逻辑-20260402.xmind`。

### 可用功率指标结构

```
可用发电功率 → 可用功率（可用电量日准确率）
├── 风电场
│   ├── 指标精度
│   │   ├── 数据颗粒度: 1秒钟1个点（计算积分电量）
│   │   ├── 指标计算逻辑: 当月每日计算
│   │   └── 指标查看逻辑: 次日查看前日
│   ├── 指标计算逻辑
│   │   └── 公式型 → 可用电量日准确率
│   │       ├── 是否限电期间
│   │       │   ├── 限电期间 → 跳过限电期间
│   │       │   └── 非限电期间 → 开始统计
│   │       │       ├── 实际发电量
│   │       │       │   ├── 未配备储能站 → 远动集电线路有功之和的绝对值
│   │       │       │   └── 配备储能站 → 集电线路有功之和 + 储能有功，取绝对值
│   │       │       └── 可用发电量 → 场站可用功率
│   ├── 双细则合格标准: 可用电量的日准确率应不小于97%
│   ├── 考核电量: 每降低1%按全场装机容量×0.02 分/万千瓦考核
│   └── 光伏电站: 风光一致
```

### 限电判断逻辑

```
AGC有功功率目标值 < 额定容量 → 限电（产生告警）
AGC有功功率目标值 = 额定容量 → 不限电
```

## 七、计算流程总览

```
calculateDay(param, dateTime):
    ├── 阶段1: 查询数据
    │   ├── 获取参数: 合格标准、装机容量、各类点号
    │   ├── 查询遥测数据(远动): 集电线路有功功率 + 储能设备有功功率 + 可用发电电力功率
    │   └── 查询遥调数据(AGC): AGC有功功率目标值
    │
    ├── 阶段2: 指标计算
    │   ├── 逐分钟判断限电:
    │   │   └── 限电 = AGC目标值 < 装机容量
    │   │
    │   ├── 遍历全天86400秒:
    │   │   ├── 实发功率 = |Σ(集电线路) + Σ(储能)|
    │   │   ├── 可用功率 = 可用发电电力功率
    │   │   ├── 每秒实发电量 = 实发功率 / 3600
    │   │   ├── 每秒可用电量 = 可用功率 / 3600
    │   │   └── IF 该秒所在分钟为非限电:
    │   │       ├── 非限电实际电量 += 每秒实发电量
    │   │       └── 非限电可用电量 += 每秒可用电量
    │   │
    │   └── 计算准确率:
    │       ├── 非限电电量均为0 → 准确率 = 100%
    │       ├── 可用电量=0, 实际电量>0 → 准确率 = 0%
    │       └── 否则 → 准确率 = 实际电量 / 可用电量 × 100%
    │
    └── 阶段3: 封装结果
        ├── value = 准确率(%)
        ├── detail = 非限电实际电量--非限电可用电量
        ├── 考核分 = (97 - 准确率) × 装机容量 × 0.002
        └── 考核状态

calculateMonth(detail):
    └── 月考核分 = Σ(每日考核分)
```

### 边界情况

| 场景 | 处理方式 |
|------|----------|
| 全天无数据 | 准确率 = 100%, 不考核 |
| 全天限电 | 准确率 = 100%, 不考核 |
| 部分分钟无数据 | 用0值填充 |
| 可用电量=0, 实际>0 | 准确率 = 0%, 考核 |
| 电量均为0 | 准确率 = 100%, 不考核 |
