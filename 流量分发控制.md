## 数据模型

```
FlowSource (流量源) ──M:N── FlowSourceSinkRelation ──M:N── TestEnvSink (测试环境)
                         │
                         └── Enabled/disabled per relation
```

### 流量源 (FlowSource)
- 数据来源: hekou, zhuyuan, qlg-tuomin 等
- 日期匹配模式、文件路径

### 测试环境 (TestEnvSink)
- 独立的测试环境配置
- RabbitMQ 连接设置

## 核心功能

1. **拓扑可视化** - SVG 流程图展示 source → target 关系
2. **流量源管理** - PCAP 数据来源配置 (日期模式、路径)
3. **测试环境管理** - 独立 RabbitMQ sink 配置
4. **关联控制** - 流量源与测试环境多对多映射
5. **调度控制** - 启动/停止定时文件处理 ()
