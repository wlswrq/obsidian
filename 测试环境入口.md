# 测试环境入口

## 概述

Hi5000 测试环境入口是 Hi5000 产品的测试环境管理门户，为运维人员提供快速访问各测试/生产/演示环境的入口。页面采用深色科技风格 UI，支持多维度筛选、环境详情查看、编辑、添加等操作。

## 页面入口

用户入口: /env/env.html

## 技术架构

- 前端: static/env/env.html (静态 HTML，AJAX 获取数据)
- 后端: EnvironmentController, EnvironmentService, Environment 实体

## API 接口

| 方法 | 端点 | 说明 |
|------|------|------|
| GET | /api/environments | 获取所有未删除环境 |
| POST | /api/environments | 创建新环境 |
| PUT | /api/environments/{id} | 更新环境 |
| DELETE | /api/environments/{id} | 逻辑删除 |

## 数据模型

### Environment 实体

| 字段 | 数据库列名 | 类型 | 说明 |
|------|-----------|------|------|
| id | - | Long | 主键 |
| name | name | String(100) | 环境名称 |
| type | type | String(20) | 应用类型：wind, solar, hybrid |
| envType | env_type | String(20) | 环境类型：test, prod, demo |
| systemType | system_type | String(20) | 系统类型：station, control |
| product | product | String(50) | 产品名称 |
| station | station | String(100) | 场站名称 |
| province | province | String(50) | 省份 |
| database_ | database_ | String(50) | 数据库类型 |
| version | version | String(50) | 版本号 |
| url | url | String(200) | 环境访问 URL |
| jenkinsUrl | jenkins_url | String(200) | Jenkins 地址 |
| qrcodeUrl | qrcode_url | String(200) | 二维码收资 URL |
| rabbitHost | rabbit_host | String(50) | RabbitMQ IP |
| rabbitVirtualHost | rabbit_virtual_host | String(50) | RabbitMQ VirtualHost |
| deleted | deleted | Boolean | 逻辑删除标记 |

## 数据库变更

环境表新增字段 DDL：

ALTER TABLE environment ADD COLUMN qrcode_url VARCHAR(200) DEFAULT NULL COMMENT '二维码收资 URL';

ALTER TABLE environment ADD COLUMN rabbit_host VARCHAR(50) DEFAULT NULL COMMENT 'RabbitMQ IP 地址';

ALTER TABLE environment ADD COLUMN rabbit_virtual_host VARCHAR(50) DEFAULT NULL COMMENT 'RabbitMQ VirtualHost';

## 核心功能

- 环境卡片展示（名称、类型、省份、版本、优先级）
- 四级筛选过滤
- 快速进入环境（点击卡片打开 URL）
- 二维码收资跳转（卡片新增图标按钮）
- Jenkins 跳转
- 环境详情查看/编辑（含 RabbitMQ 配置）
- 添加新环境

## 版本查询说明

版本号从数据库 version 字段读取。版本查询 API 地址由测试环境 url 拼接 /system/version 接口，POST 请求，无参数。

版本查询示例：
- 环境 URL: http://192.168.40.39:9000
- 完整 API: http://192.168.40.39:9000/system/version
